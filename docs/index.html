Quick setup

WhyLowDPS is distributed as a Windows desktop app. Start credential-free, or connect your own Battle.net API client for account-aware features.

    Get the latest Windows release from GitHub.

  3. 3
    Start comparing

    Paste SimC data or import a connected character, then run a simulation.

Local by design

Your credentials stay on your machine.

Your Battle.net API credentials are stored locally and are not uploaded or shared by WhyLowDPS. There is no custom WhyLowDPS account backend.

The app uses Blizzard APIs for game data, GitHub for releases and published data updates, and Raidbots-related data where simulation workflows require it.
