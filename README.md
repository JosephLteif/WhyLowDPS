# WhyLowDPS

**WhyLowDPS** is a Windows desktop app for World of Warcraft players who want a more personal, local-first way to simulate, track, and improve their characters.

The app is built around the player and their characters. Its goal is to bring common character-optimization workflows into one place: local simulations, sim history, route-based sims, gear planning, weekly game data, wishlists, and other tools that help you make better upgrade decisions without constantly jumping between different websites or reopening the game.

## What it does

WhyLowDPS lets you simulate your WoW character locally using SimulationCraft, track live sims and previous sim results, save routes from SimC strings, reuse saved character SimC data, and quickly check useful character information such as the Great Vault.

It also includes several optimization modes, such as consumable comparisons, tier slot planning, trinket combination testing, crest and upgrade planning, drop finding, and character-specific item wishlists.

The app also provides current-season information such as dungeons, raids, weekly affixes, and other game data that is useful when planning upgrades or simulations.

## Main features

- Local character simulation using SimulationCraft
- Live simulation tracking
- Simulation history
- Saved SimC strings per character
- Route saving from SimC strings for route-specific sims
- Great Vault checking directly from the app
- Current season dungeon, raid, and affix information
- Consumable Matrix for comparing consumables
- Tier Slot Matrix for planning tier conversions
- Trinket Matrix for testing trinket combinations
- Upgrade and crest planning
- Drop finder
- Character-based wishlist groups for tracking desired items

There are also additional modes and features that are not fully listed here yet, as the project is still evolving.

## Data sources and privacy

WhyLowDPS currently uses Blizzard APIs as the main source for game and character data. It also uses GitHub for application updates and release-published data refreshes, and Raidbots-related data where needed for consumables, items, and simulation-related functionality.

The app requires your own Battle.net API credentials to function. These credentials are generated by you through the Battle.net developer portal, following the instructions shown inside the app.

Your Battle.net API credentials are stored locally on your machine only. They are not shared, uploaded, or sent to any custom backend by the application.

## Platform support

WhyLowDPS currently supports **Windows only**. There are no current plans to support additional platforms.
