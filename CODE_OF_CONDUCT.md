# Code of Conduct

## Our standard

This project expects respectful, professional collaboration.

Examples of expected behavior:

- Be respectful and constructive.
- Focus feedback on ideas and implementation, not people.
- Assume good intent while still being direct and precise.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Trolling or deliberate disruption
- Publishing private information without consent

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove or reject contributions that violate it.

## Reporting

Report behavior issues privately to the maintainer.
