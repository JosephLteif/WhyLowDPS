Local by design

Your credentials stay on your machine.

Your Battle.net API credentials are stored locally and are not uploaded or shared by WhyLowDPS. There is no custom WhyLowDPS account backend.

The app uses Blizzard APIs for game data, GitHub for releases and published data updates, and Raidbots-related data where simulation workflows require it.

FAQ

Answers before you download.

Do I need a Battle.net API key to use the app?

No. Light Mode is a first-class workflow and lets you use supported local simulation and planning features without credentials.

What does Light Mode include?

Light Mode supports manual SimC-based workflows, local analysis, saved history, upgrade planning, and route, raid, and dungeon reference tools.

What do credentials unlock if I add them later?

Connected Mode adds Battle.net-backed character context like saved rosters, Great Vault integration, wishlists, and character-aware tooling.

Does WhyLowDPS run simulations locally?

Yes. The app runs SimulationCraft locally on your Windows machine rather than sending your simulation jobs through a separate hosted queue.

Is the app free and Windows-only?

Yes. The landing page and release flow are aimed at Windows desktop users, and the app is distributed through GitHub releases.

Is this affiliated with Blizzard, SimulationCraft, or Raidbots?

No. WhyLowDPS is an independent project and is not affiliated with Blizzard Entertainment, SimulationCraft, or Raidbots.
